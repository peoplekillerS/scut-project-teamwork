解压taxi_analysis_datastructure-1.0压缩包后，
进入其文件中，在上面的文件路径那一栏输入CMD，
进入   相应的目录    的命令行后，
运行：
python setup.py install
其会下载和检查相应的依赖。
之后运行：
python ui_test.py
这个文件是核心程序，其初始化有一点久，因为读文件费时。之后会跳出GUI界面，便可进行相应操作。



题外话：
我们使用的是c++、python交互的方式来实现
pyQt5作为前端，c++实现后端，再把c++文件封装成dll库，然后通过import导入，使用了各种可视化的包，实现了全部功能，只有锦上添花没有美中不足，要用要自行申请百度地图的api，还转化了加密坐标系
